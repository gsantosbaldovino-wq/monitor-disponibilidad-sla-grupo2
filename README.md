# monitor-disponibilidad-sla-grupo2
Sistema de monitoreo de disponibilidad y SLA con registro de uptime, downtime y análisis de servicios - Grupo 2
Proyecto académico desarrollado para monitorear servicios web mediante:

- Registro de uptime y downtime
- Métricas de disponibilidad SLA
- Verificación de servicios
- Uso de python-nmap para monitoreo de puertos
